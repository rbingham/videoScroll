# videoScroll
[![Bower version](https://img.shields.io/badge/bower-v-green.svg)] (https://github.com/rbingham/playVideo)

Video Scroll is an angular directive to allow for playing an html5 video while you scroll.

## Code Examples

```html
<video play-video>

</video>
```
